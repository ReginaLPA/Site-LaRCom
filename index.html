Nossos Projetos




X-Saber

O X-Saber é uma plataforma completa para distribuição e aplicações de conteúdo pedagógico, personalizadas através da fábrica de jogos educacionais do LarCom.

SIGM 4.0

O SIGM 4.0 é um projeto de desenvolvimento de um ERP/CRM especializado para a gestão pública a partir da plataforma, open source, de ERP/CRM "Dolibarr".
A solução customizada SIGM 4.0 possui também um módulo dedicado de Inteligência Analítica para tratamento de dados usando técnicas de Inteligência Artificial. Neste projeto, a partir das necessidades apresentadas pelos stakeholders municipais, nossa equipe estuda as características da demanda e personaliza e/ou desenvolve códigos para o atendimento dessas necessidades a partir da plataforma. Este é um projeto totalmente orientado à Cloud e aderente com os requisitos da LGPD.

SmartNet

Resultado de uma dissertação de mestrado desenvolvida na FEEC/UNICAMP, a SmartNET é uma plataforma de rede resiliente populada com agentes inteligentes de monitoramento 24x7 em regime autonômico e por sala de situação.
A SmartNET foi desenvolvida para permitir a implementação de Cidades Inteligentes de baixo custo e alto desempenho.
